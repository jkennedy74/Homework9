Why laTTitude?

The purpose of laTTitude is to analyze how weather changes with proximity to the equator.

To accomplish this, we first pulled data fromm the OpenWeatherMap API to assemble a dataset on over 500 cities. After assembling the dataset, we used Seaborn and Matplotlib to plot various aspects of the weather verses the city's latitude. Factors that were included: Temperature Cloudienss Wind Speed Humidity This site provides the source data and visualizations created as part of the analysis as well as explainations and descriptions of any trends and correlations witnessed.
