How is weather impacted by proximity to the equator?

Its common knowledge that weather is warmer near the equator. This is because it recieves more direct exposure to sunlight than other areas of the planet such as the poles. In what other ways does distance from the equator affect the weather? In this exercise, we will explore the weather of 4000 randomly selected cities and plot the following against the latitude:

  • Maximum Temperature
  • % Humidty
  • % Cloudiness
  • Wind Speed

The purpose of this project is to build a dashboard leveraging what we've learned about HTML, CSS, and JS. In exploring the effect proximity to the equator has on different aspects of the weather, we will utilize the following technologies as well:

  • OpenWeatherMap API
  • Python (Pandas, Matplotlib, Numpy, CityPy, and Seaborn)
  • Bootstrap 4.1.2
  • Bootswatch1
