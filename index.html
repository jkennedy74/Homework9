laTTitude

latitude with attitude

The purpose of this project is to analytize how the weather changes as proximity to the equator descreases. To accomplish this, we first pulled data fromm the OpenWeatherMap API to assemble a dataset on over 500 cities. After assembling the dataset, we used Matplotlib to plot various aspects of the weather verses the city's latitude. Factors that were included: Temperature Cloudienss Wind Speed Humidity This site provides the source data and visualizations created as part of the analysis as well as explainations and descriptions of any trends and correlations witnessed.
